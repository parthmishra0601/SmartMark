# SmartMark

SmartMark is a bookmark management web application where users can securely save, view, and delete bookmarks for books of any genre. The application is built using Next.js for the frontend, TailwindCSS for styling, and Supabase for database and authentication.

## Features
- User authentication
- Add bookmarks (name, URL, description)
- View saved bookmarks
- Delete bookmarks
- Realtime updates

## Tech Stack
- Next.js
- Supabase
- TailwindCSS

## Getting Started

Run the development server:

npm run dev

Open http://localhost:3000 in your browser to view the app.

## Deployment
The application can be deployed using Vercel.
